# PostgreSQL Project: Amazon Sales Data Analysis

## Overview
This project demonstrates the use of **PostgreSQL** for managing and analyzing Amazon sales data. It includes SQL scripts for creating tables, inserting data, and performing queries to extract meaningful insights such as sales performance, product trends, and customer behavior.

## Project Contents
- `e14d11d8-ff26-4ec9-b8e1-884d8747d203.sql`: Main SQL script containing:
  - Table creation scripts
  - Data insertion queries
  - Sample queries for analysis

## Key Features
- **Database Design:** Proper relational tables for products, sales, customers, and orders.
- **Data Analysis Queries:** Aggregate functions, joins, filtering, and sorting to derive insights.
- **Scalable & Modular:** Easy to extend for additional analysis or larger datasets.

## Tools & Technologies
- **PostgreSQL:** Used for database creation and management.
- **SQL Queries:** For CRUD operations and data analysis.
